# Changelog

All notable changes to InvToolkit will be documented in this file.

## [0.1.0] — 2026-03-02

### Added
- Initial Flask + yfinance dashboard with 19 tabs
- 60+ API endpoints with CRUD operations
- Full data import from Google Sheets Excel export
- Dividend Log calendar matrix with collapsible years and inline editing
- Monthly Data with auto-linked dividend income from Dividend Log
- Annual Data computed from Monthly Data + Dividend Log
- My Lab: 12 investor portfolios with allocation charts and research button
- Notes box per portfolio (last update + source URL)
- Intrinsic Values list with sector, category, signal badges, div yield, P/E
- Super Investor Buys (50 stocks with summary stats)
- Salary dashboard with tax breakdown, retirement plan, salary history
- Cost of Living comparison (26 cities, downtown/suburban)
- Passive Income tracking with stacked bar chart
- Portfolio Projections with configurable parameters
- Rule of 4% retirement scenarios (20/30/40 years)
- Dividend Yield column in Positions tab
- Cross-platform launchers (macOS, Windows)
- Google Drive auto-detection for data sync between machines
- Dark theme UI with Chart.js visualizations
- 5-minute cache with disk persistence
